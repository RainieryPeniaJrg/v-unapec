# Materiales de Laboratorio - Referencias

Los materiales de laboratorio originales se encuentran en la carpeta raíz `Contexto/`:

```
Contexto/
├── Lab-Tema02-ISO410.pdf   (Laboratorio Tema 02 - Verificación vs Validación)
└── Lab-Tema03-ISO410.pdf   (Laboratorio Tema 03 - Testing Avanzado)
```

## Cómo Acceder

### Opción 1: Desde la raíz del proyecto
```
V UNAPEC/
└── Contexto/
    ├── Lab-Tema02-ISO410.pdf
    └── Lab-Tema03-ISO410.pdf
```

Abre los PDFs directamente desde `Contexto/`.

### Opción 2: Referencias en esta carpeta
Este directorio (`docs/contexto/`) contiene documentación complementaria:
- `README.md` - Índice y guía de navegación
- `referencias/` - Links útiles y estándares
