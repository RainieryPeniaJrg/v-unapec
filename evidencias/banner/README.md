# Evidencias Banner E2E

Esta carpeta centraliza evidencias de la automatizacion real de Banner:

- `screenshots/`: capturas de post-login, horario y post-logout.
- `logs/banner-runs.jsonl`: log estructurado por corrida.

Ejecutar:

```powershell
npm run test:banner:headed
```

Luego abrir reporte:

```powershell
npm run test:banner:report
```
